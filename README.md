# AI Personal Assistant (Google ADK)

This project is an AI-powered personal assistant built using Google Agent Development Kit (ADK) and deployed on Cloud Run.

## Features
- Answers user queries using Gemini model
- Interactive chat UI
- Deployed on Google Cloud

## Tech Stack
- Python
- Google ADK
- Vertex AI (Gemini)
- Cloud Run

## Project Highlights
- Built using Google ADK
- Uses Gemini (LLM)
- Deployed on Cloud Run
- End-to-end AI application
- 
## How to Run
1. Install dependencies
2. Configure .env
3. Run agent locally or deploy via ADK CLI

## Example Queries
- "Where are polar bears located?"
- "What animals are in the zoo?"
- "Tell me about lions"
